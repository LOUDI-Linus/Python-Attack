面向过程(函数)：根据操作数据实现某一个功能。=> 强调实现功能
面向对象(类)：把数据和功能封装在一起，称之为对象 => 强调对象(谁去做这件事)
 人=>[数据(姓名、年龄、性别),功能(上网)]=>对象


对象包含两个部分：
    数据 => 变量
    方法 => 函数 def => 一般在类中定义的函数称为方法

类 class => 描述某类物品的特征
   信息 => 数据 => 变量
   行为 => 函数

父类 -> 子类 / 基类 -> 派生类 (类的继承)
    车 -> 电瓶车 -> 爱玛 -> .....

class 电瓶车():
    10个方法
    所有电瓶车的特点....

class 爱玛电瓶车(电瓶车):
    1个方法
    定义爱玛电瓶车自己独有的特点即可

class 电瓶车():
    10个方法
    所有电瓶车的特点....

    11个方法
    定义爱玛电瓶车自己独有的特点即可
