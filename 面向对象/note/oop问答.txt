1. metaclass作用？以及应用场景？
	metaclass是元类，创建类的类。
	定制创建类的过程 => 添加一些自定义的逻辑
	   foo => 阻止 
   	   FOO => foo
   	   类中的属性名不能包含两个连续的下划线
   	默认元类是type


2. 魔法函数__call__怎么使用?
	class A():pass
	a = A()
	当调用某个对象的时候，需要给它加上__call__方法	
	a()
	__add__ => 对象需要进行+运算的时候， __add__


3. staticmethod、classmethod, property的用法及功能
   装饰器：=> @名称
   staticmethod：把类的一个方法封装成静态方法（）
   classmethod：把类的一个方法封装成类态方法（cls）
   property：把类的一个方法封装属性


4. 类属性与实例属性是什么？区别和应用场景？
	类属性 => 多个实例共用一个属性
	实例属性 => 每个实例自己拥有一个属性值


5. python经典类与新式类的MRO有什么区别
	经典类：DFS
	新式类：C3


6. 如何判断一个对象是函数还是方法？函数与方法的区别
	函数: 与实例和类没有绑定关系
	方法: 与实例和类有绑定关系
